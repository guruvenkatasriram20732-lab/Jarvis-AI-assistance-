# 🤖 Jarvis AI Voice Assistant

Jarvis AI is a Python-based desktop voice assistant that performs various daily tasks through voice commands. It features a simple GUI, speech recognition, text-to-speech, weather updates, media playback, system controls, and web automation.

## ✨ Features

- 🎙️ Voice command recognition
- 🔊 Natural Text-to-Speech (Male Voice)
- 🖥️ CustomTkinter GUI
- 🌦️ Live weather updates
- 🎵 Play songs on YouTube
- 🌐 Open Google and YouTube
- 🎧 Open Spotify
- 📝 Open Notepad
- 🔉 System volume control
  - Volume Up
  - Volume Down
  - Mute
- ⏰ Tell current time
- 📚 Wikipedia search
- 💻 Shutdown and Restart computer
- 🚪 Exit using voice command

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Jarvis-AI.git
```

### 2. Navigate to the project folder

```bash
cd Jarvis-AI
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python jarvis.py
```

## 🎤 Sample Voice Commands

| Command | Action |
|---------|--------|
| Hello Jarvis | Greets the user |
| Play Believer | Plays song on YouTube |
| What is the time | Speaks current time |
| Who is Elon Musk | Searches Wikipedia |
| Open Google | Opens Google |
| Open YouTube | Opens YouTube |
| Open Spotify | Opens Spotify |
| Open Notepad | Opens Notepad |
| Weather | Provides weather information |
| Volume Up | Increases system volume |
| Volume Down | Decreases system volume |
| Mute Volume | Mutes system volume |
| Shutdown Computer | Shuts down the PC |
| Restart Computer | Restarts the PC |
| Exit | Closes Jarvis |

## 🔑 Configuration

The project uses the OpenWeather API for weather information.

Replace the API key in `jarvis.py` with your own API key:

```python
api_key = "YOUR_API_KEY"
```

You can get a free API key from:

https://openweathermap.org/api
